# 🌸 Anime Character Visual Explorer

## 📌 Project Description

Anime Character Visual Explorer is a web application that allows users to search and explore detailed information about anime characters. Users can view character images, basic details, and the anime in which the character appears. The application provides an interactive and user-friendly interface to enhance the exploration experience.

---

## 🎯 Purpose of the Project

The goal of this project is to demonstrate understanding of:

* JavaScript fundamentals
* API integration using fetch
* Dynamic UI rendering
* Use of array higher-order functions (map, filter, sort)

---

## 🌐 API Used

This project uses the **Jikan API**, an unofficial MyAnimeList API.

Example endpoint:
https://api.jikan.moe/v4/characters?q=naruto

---

## 🚀 Features (Planned)

### 🔍 Search

Users can search for anime characters by name.

### 🎯 Filtering

Users can filter characters based on criteria such as popularity or anime.

### 🔄 Sorting

Characters can be sorted alphabetically or by popularity.

### 🖼️ Image Gallery

Displays character images in a visually appealing format.

### ❤️ Favorites

Users can save their favorite characters using local storage.

### 🌙 Dark Mode

Toggle between light and dark themes.
